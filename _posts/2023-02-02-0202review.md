### before(), after()
before() - 요소 앞에 요소 추가 <br/>
after() - 요소 뒤에 요소 추가 <br/>
<br/>

#### 예제 1
```jsp
<%@ page language="java" contentType="text/html; charset=UTF-8"
    pageEncoding="UTF-8"%>
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Insert title here</title>
	<script src="http://code.jquery.com/jquery-latest.js"></script>
	<script>
		$(function()
		{
			$("#bb").before("하나");
			$("#bb").after("셋");
		});
	</script>
</head>
<body>
<span id="bb"> 둘 </span>
</body>
</html>
```
결과 - 하나 둘 셋 <br/>
<br/>

참고 예제) css 가상요소 ::before와 ::after
```jsp
<%@ page language="java" contentType="text/html; charset=UTF-8"
    pageEncoding="UTF-8"%>
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Insert title here</title>
	<style>
		#aa::after {
			content:"33";
		}
		#aa::before {
			content:"11";
		}
	</style>
</head>
<body>
	<span id="aa"> 22 </span>
</body>
</html>
```
결과 - 11 22 33
<br/>

#### 예제2

```jsp
<%@ page language="java" contentType="text/html; charset=UTF-8"
    pageEncoding="UTF-8"%>
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Insert title here</title>
	<script src="http://code.jquery.com/jquery-latest.js"></script>
	<script>
		$(function()
		{
			$("#cc").click(function()
			{
				$("#cc").after($("#aa"));
			});
		});
	</script>
</head>
<body>	<!-- jquery_after2.jsp -->
	<div id="aa">
		<span id="bb"> Hello </span> <b> 안녕 </b>
	</div>
	<hr>
	<div id="cc"> 클릭해보세요 </div>
</body>
</html>
```
